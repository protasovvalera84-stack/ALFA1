Редактор robots.txt

Первый файл, который читает поисковый бот. Ошибки — самая частая причина проблем с индексацией.

Важно (2026): Не закрывайте CSS и JS-файлы — иначе Googlebot рендерит сайт как сломанный. Убедитесь, что путь к Sitemap: указан верно.

Быстрые шаблоны:

Schema.org JSON-LD

Структурированные данные для расширенных сниппетов в Google и Яндекс (+15-30% CTR).

LocalBusiness ✓ (встроен)

Компания, адрес, телефон, часы работы. Показывается в блоке с картой в Google.

SEO-чеклист 2026

Технические требования Google и Яндекс. Отмечайте выполненные пункты.

Core Web Vitals (Google + Яндекс)
LCP ≤ 2.5 сек
Go SSR обеспечивает TTFB ~30ms — LCP будет отличным автоматически.
INP ≤ 200 мс (заменил FID с марта 2024)
Минимум тяжёлого JS. Все обработчики лёгкие — требование выполнено.
CLS ≤ 0.1
Фиксированные размеры всех блоков — нет сдвигов при загрузке.
Индексация и структура

Серверный маршрут зарегистрирован, редактируется выше.

Open Graph теги для соцсетей

GEO — Generative Engine Optimization (2026)

В 2026 году ~18% коммерческих поисковых запросов генерируют AI-ответы (Google AI Overview, Perplexity, ChatGPT). Для попадания в них: используйте структурированные данные Schema.org, пишите контент в формате «вопрос-ответ» (BLUF), обеспечьте server-side rendering без зависимости от JavaScript — ваш сайт уже соответствует этим требованиям благодаря Go SSR.
