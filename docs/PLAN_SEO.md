## Часть 1: Техническое SEO (приоритет #1)

### 1.1 Core Web Vitals (2026)
Обязательные метрики по стандартам Google и Яндекс:

| Метрика | Норма | Наш сайт (цель) |
|---------|-------|----------------|
| LCP (загрузка) | ≤ 2.5 сек | ~0.8 сек (Go SSR) |
| INP (отклик) | ≤ 200 мс | ~50 мс (минимум JS) |
| CLS (стабильность) | ≤ 0.1 | ~0.02 (фиксированные размеры) |
| TTFB | ≤ 600 мс | ~30 мс (Go compile) |

**Действия:**
- [ ] Проверить в PageSpeed Insights: https://pagespeed.web.dev/
- [ ] Проверить в Google Search Console → Core Web Vitals
- [ ] Зарегистрировать в Яндекс Вебмастер → Технические проблемы

### 1.2 Индексация

**robots.txt (оптимальный для Google + Яндекс 2026):**
```
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /api/

User-agent: Googlebot
Allow: /

User-agent: YandexBot
Allow: /

Sitemap: https://ВАШИ_ДОМЕН/sitemap.xml
```

**Важно:** НЕ закрывать CSS и JS — поисковики рендерят страницы!

### 1.3 Sitemap.xml
- Генерируется автоматически сервером Go при изменении контента
- Включает: главную страницу, страницы услуг, историю, контакты
- `<lastmod>` обновляется при изменении контента
- Отправить в Google Search Console и Яндекс Вебмастер
